# 🫀 Clasificación de Enfermedad Cardíaca con Machine Learning

Proyecto de aprendizaje automático enfocado en la **predicción de enfermedades cardíacas** a partir de datos clínicos, utilizando técnicas de análisis de datos y modelos de clasificación.

---

## 📌 Descripción

Este proyecto tiene como objetivo desarrollar un modelo de Machine Learning capaz de predecir la presencia de enfermedad cardíaca en pacientes, a partir de variables clínicas como edad, presión arterial, colesterol, entre otras.

El análisis se realiza utilizando un notebook en Python, donde se incluyen las etapas de:

* Exploración de datos (EDA)
* Preprocesamiento
* Entrenamiento del modelo
* Evaluación de resultados

---

## 📊 Dataset

El dataset utilizado proviene de la plataforma **Kaggle** y contiene información médica relevante para el diagnóstico de enfermedades cardíacas.

Características típicas del dataset:

* Edad
* Sexo
* Tipo de dolor en el pecho
* Presión arterial
* Colesterol
* Frecuencia cardíaca máxima
* Presencia de enfermedad (variable objetivo)

---

## ⚙️ Tecnologías utilizadas

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn
* Jupyter Notebook (Google Colab)

---

## 🧠 Modelo de Machine Learning

Se implementa un modelo de clasificación (por ejemplo, Regresión Logística), el cual:

* Divide los datos en entrenamiento y prueba
* Aprende patrones a partir de los datos
* Predice la probabilidad de enfermedad cardíaca

---

## 📈 Resultados

El modelo es evaluado utilizando métricas como:

* Accuracy
* Matriz de confusión
* Precisión y recall
